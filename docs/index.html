Server Events

inventory:server:OpenInventory
Server event when a player wants to open an inventory
  • name string the inventory type (Config.InventoriesType)
  • id number the inventory id
  • other Inventories[][] the opened inventory array (optionnal) (optional)
inventory:server:SaveInventory
A server event to save the specified inventory name
  • name string The inventory name (in the config)
  • id number The inventory id
inventory:server:SetIsOpenState
A server event handler to let the client set an inventory closed state
  • isOpen bool Is the new inventory state open?
  • name string The inventory name
  • id number The inventory id
inventory:server:addTrunkItems
⚠️ DEPRECATED! Do not use it.
Used to be the event to set the items in a trunk.
