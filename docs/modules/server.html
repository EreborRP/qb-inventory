A server event to save the specified inventory name
  • name string The inventory type (in the config)
  • id number The inventory id
inventory:server:SetInventoryData
Move an item from one inventory to another
  • fromInventory string The inventory to move the item from (Config.InventoriesType)
  • toInventory string The inventory to move the item to (Config.InventoriesType)
  • fromSlot number The slot to move the item from
  • toSlot number The slot to move the item to
  • amount number The amount of the item to move (if not specified, move all) (optional)
inventory:server:SetIsOpenState

Server event when a player wants to craft an item from a crafting table

Parameters:

  • amount number The amount of items crafted
  • toSlot number The slot where the crafted item is put
